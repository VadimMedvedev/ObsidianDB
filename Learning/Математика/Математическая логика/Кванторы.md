[[Переменные]] можно связывать, не только фиксируя их. Для этого ещё можно исользовать кванторы "для всех" и "для некоторого". 
Например высказывание:
$$(x+1)^2=x^2+2x+1$$
имеет одну свободную переменную $x$ и меняет свою истенность в зависимости от неё (т.е. не обязательно имеет однозначное истиностное значение). Но теперь:
$$(x+1)^2=x^2+2x+1\ \ \ \ \ \ \ для\ всех\ x$$
высказывание имеет одну связанную переменную, и следственно имеет одно конкретное истенностное значение (в нашем примере высказывание истенно, так как для всех $x$ утверждение верно).
Высказывание:
$$x+3=5 \ \ \ \ для\ некоторого\ x$$
тоже имеет одну связанную переменную, и не имеет свободных переменных, следовательно может быть истенным или ложным. Т.к. существует $x=2$ при котором равенство верно, то всё высказывание с квантаром верно.
Но высказывание:
$$x+3=5 \ \ \ \ для\ всех\ x$$
уже не будет верно (наприер для $x=3$ равенство не выполняется).

**Квантор всеобщности.** Пусть $P(x)$ - [[Высказывания|высказывание]] зависящие от свободной переменной $x$. Тогда высказывание "$P(x)$ правдиво для всех $x$ типа $T$" истенно, если для всех $x$ типа $T$ высказывание $P(x)$ всегда становится истенным.

Тем самым, чтобы доказать истенность высказывания со свзяанной квантором всеобщности переменой, надо для всех $x$ типа $T$ без различия доказать, что $P(x)$ верно. Чтобы доказать ложность, достаточно найти контрпример, при каком $x$ типа $T$ высказывание $P(x)$ становится ложным.
Но высказывание "$P(x)$ правдиво для всех $x$ типа $T$" становится автоматически истенным, когда не существует объектов типа $T$ (действительно, ведь нельзя и привести контпримеров). Например таким образом истенным становится высказывание:
$$6<2x<4\ \ \ \ \ \ \ \ для\ всех\ 3<x<2$$
Квантор всеобщности обозначают $\forall$ и заменяют им слова "для всех", например:
$$\forall x\in X\ \ \ \ \ \ x+3=5$$

Квантор всеобщности, как следствие, позволяет нам подставлять любые $x$ типа $T$ в высказывание связанное им, и быть уверенным, что высказывание будет верным. Например "$(x+1)^2=x^2+2x+1$ для всех вещественных числе $x$" говорит, что мы можем взять, например, $\pi$ в качестве $x$ и высказывание $(\pi+1)^2=\pi^2+2\pi + 1$ будет верным.

**Квантор существования.** Высказывание "$P(x)$ правдиво для некоторого $x$ типа $T$" истенно, когда существует такой $x$ типа $T$, удволетворяющий высказыванию $P(x)$.

Для доказательства такого высказывания, достаточно привести конкретный $x$ типа $T$ при котором высказывание $P(x)$ истенно (хотя их может быть и больше чем один). Для доказательство ложности в таком случае, надо доказать что для всех $x$ типа $T$ высказывание не выполняется.
В отличие от квантора всеобщности, если $T$ не имеет объектов в себе, то высказывание с квантором всеобщности будет автоматически ложным (нельзя найти удовлетворяющего утверждению $x$). Например ложно:
$$6<2x<4\ \ \ \ \ \ \ \ существует\ 3<x<2$$
Для квантора всеобщности используется обозначение $\exists$, заменяющее слово "существует". Пример:
$$\exists x\in X\ \ \ \ \ \ x+3=5$$

Квантор существования не такой гибкий как квантор всеобщности, но он позволяет с увереностью сказать, что $x$ типа $T$ удовлетворяет высказыванию связанному этим квантором, хотя, мы даже можем и не знать, какой конкретно $x$ ему удовлетворяет.
