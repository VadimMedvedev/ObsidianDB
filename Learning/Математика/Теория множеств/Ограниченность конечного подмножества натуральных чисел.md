**Лемма.** Если $n$ - [[Построение натуральных чисел. Аксиомы Пеано|натуральное число]], либо ноль, и $f:\{i\in\mathbb N\ |\ i\leqslant n\}\longrightarrow \mathbb N$ - некоторая [[Определение функции|функция]], то существует некоторое натуральное число $M$ такое, что $f(i)\leqslant M$ для всех $i\leqslant n$. 
**Доказательство.** Если $n=0$, то очевидно $M$ можно взять любое, потому что оно будет больше всех элементов пустого множества.
Для того, чтобы доказать лемму для всех $n$ - натуральных, воспользуемся [[Доказательства по индукции|индукцией]] по $n$. Если $n=1$, то возьмём $M=f(1)$. Тогда $\forall i\leqslant 1$ будет выполнятся $f(i)\leqslant M$ (ведь $i$ может быть равен только $1$). Пусть теперь для $n$ лемма доказана, докажем для $n\smash+\smash+$. Сузим область определения функции $f$ с $N_{++}=\{i\in\mathbb N\ |\ i\leqslant n\smash+\smash+\}$, до $N=\{i\in\mathbb N\ |\ i\leqslant n\}$ композицией c [[Отображение включения и идентичное отображение|отображением включения]] $f\circ id_{N\to N_{++}}:N\longrightarrow\mathbb N$. По индукции, для данной функции существует $M'$ такое, что $(f\circ id_{N\to N_{++}})(i)\leqslant M'$ для всех $i\leqslant n$. Теперь, в силу [[Порядок натуральных чисел|трихотомии порядка]], либо $f(n\smash+\smash+) \leqslant M'$, либо $f(n\smash+\smash+) > M'$, в первом случае возьмём $M=M'$, а во втором $M=f(n\smash+\smash+)$. Убедимся, что $f(i)\leqslant M$ для всех $i\leqslant n\smash+\smash+$. Если $i\leqslant n$, то $f(i)=(f\circ id_{N\to N_{++}})(i)\leqslant M'\leqslant M$, если $i=n\smash+\smash+$, то аналогично $f(n\smash+\smash+)\leqslant M$. 

Если у нас есть некоторое [[Конечные и бесконечные множества|конечно множество]] $X$ содержащее в качестве элементов натуральные числа ($X\subset\mathbb N$), то, в силу его конечности, оно имеет [[Мощности множеств|натуральную мощность]] $n$, а значит существует биекция $f:\{i\in\mathbb N\ |\ i\leqslant n\}\longrightarrow X$, которую легко достроить до $id_{X\to\mathbb N}\circ f:\{i\in\mathbb N\ |\ i\leqslant n\}\longrightarrow\mathbb N$. Теперь, пользуясь леммой, мы можем сказать, что $f(1),...,f(n)\in X$ можно ограничить $M$. Таким образом, любое конечно подмножество натуральных чисел можно ограничить.
