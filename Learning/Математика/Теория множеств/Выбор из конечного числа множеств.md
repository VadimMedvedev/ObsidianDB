Ранее, уже обсуждался [[Выбор из множества|выбор элемента из одного множества]], теперь же, мы расширим наши возможности до выбора из нескольких множеств сразу.

**Лемма (Конечный выбор).** Пусть $n$ - натуральное число, и для каждого $1\leqslant i \leqslant n$ существует $X_i$ - непустое множество. Тогда существует [[Упорядоченные наборы|набор]] из $n$ элементов $(x_1,\ ...,\ x_n)$ такой, что $x_i\in X_i$ для всех $1\leqslant i\leqslant n$.
**Доказательство.** Воспользуемся для доказательства [[Доказательства по индукции|индукцией]] по $n$.
Если $n=1$, то мы можем использовать [[Выбор из множества|лемму для выбора из множества]], и тогда, раз по условию леммы $X_1$ - не пусто, то $\exists x_1\in X_1$, и значит мы нашли нужный нам упорядоченный набор $(x_1)$. 
Доказав базу, докажем теперь и шаг индукции. Пусть для $n$ лемма верна, докажем её и для $n\smash+\smash+$. Пусть $X_1,\ ...,\ X_{n\smash+\smash+}$ - непустые множества, тогда для первых $n$ множеств, по предположению индукции, существует упорядоченный набор $(x_1,...,x_n)$ такой, что $x_i\in X_i$ для всех $1\leqslant i\leqslant n$. Т.к. $X_{n\smash+\smash+}$ - тоже не пустое множество, то по [[Выбор из множества|лемме]] существует $x_{n\smash+\smash+}\in X_{n\smash+\smash+}$. Теперь мы можем составить упорядоченный набор из $n\smash+\smash+$ элементов $(x_1,...,x_n,x_{n\smash+\smash+})$ такой, что $x_i\in X_{i}$ для всех возможных $i$. Это заканчивает индукцию.

Другими словами, эту лемму можно переформулировать так: *Если каждое множество $X_i$ - [[Пустое множество и одноэлементные множества|не пусто]], то их [[Декартово произведение набора множеств|декартово произведение]] тоже не пусто*.

Кажется, что эту лемму можно без труда распространить и на выбор из бесконечного числа множеств, но на самом деле, это требует введение новой аксиомы.
