**Определение.** [[Определение функции|Функция]] $f$ называется сюръективной, если для каждого элемента из $Y$ существествует элемент из $X$, который переводится под дейтсвием функции в изначальный элемент.
$$\forall y\in Y\ \ \exists x\in X\ \ \ f(x)=y$$

Например квадратичная функция $f: \mathbb Z\longrightarrow \mathbb Z$ не является сюръективной (т.к. отрицательные числа не покрываются образом $\mathbb Z$). Если сузить область значений до $\mathbb N\cup \{0\}$, то функция станет сюръективной.

**Лемма.** [[Композиция функций|Композиция]] сюръективных функций - сюръекция.
**Доказательство.** Пусть $f: X\longrightarrow Y$ и $g: Y\longrightarrow Z$ - сюръективные функции. Их композицией будет функция $f\circ g: X\longrightarrow Z$. Покажем, что для получившийся функции, для любого $z\in Z$ найдётся свой $x\in X$ такой, что $(g\circ f)(x)=z$. Возьмём произвольный $z\in Z$, по сюръективности функции $g$, найдём такой $y\in Y$, что $g(y)=z$. Теперь по сюръективности функции $f$, найдём для этого $y$ такой $x\in X$, что $f(x)=y$. Тогда теперь, мы можем написать, что $(g\circ f)(x)=g(f(x))=g(y)=z$. Таким образом, для каждого $z\in Z$  мы можем найти необходимый $x\in X$.
