
**Определение.** Векторным (или линейным) пространством над [[Поля|полем]] $K$ множество $V$ с операциями сложения и умножения на элементы $K$, обладающие следующими свойствами:
1) Относительно сложения $V$ - [[Абелевы группы|абелева группа]];
2) $\forall \lambda\in K\ \ \forall a,b\in V:\ \ \lambda(a+b)=\lambda a+\lambda b$;
3) $\forall \lambda,\mu\in K\ \ \ \forall a\in V:\ \ \ \lambda(\mu a)=(\lambda\mu)a$;
4) $\forall \lambda,\mu\in K\ \ \ \forall a\in V:\ \ \ (\lambda+\mu) a=\lambda a+ \mu a$;
5) $\forall a\in V:\ \ \ 1\cdot a=a$.
Элементы $V$ мы будем называть векторами, а элементы $K$ - числами (хотя они могут ими и не являться). При произведении вектора на число мы всегда пишем число слева а вектор справа, таким образом не возникает путанице при использовании $0\in K$ и $0\in V$ (они могут отличаться).

В качестве примеров можно привести $\mathbb R^n$ над полем $\mathbb R$ с операцией по компонентного сложения векторов и умножения на число $\mathbb R$ отдельно по каждой компоненте. Или же например множество функций определеённых на некотором множестве $X$ со значениями в поле $K$, тогда сложение функций это функция сопоставляющее элементу сумму значений этих функций, а умножение на элемент из $K$ функция которая возвращает значение функции умноженное на это число.

Можно выписать следующие элементарные следствия из определения, которые в сущности будут повторять аналогичные следствия для [[Абелевы группы|групп]] и [[Кольца|колец]] и будут иметь такое же доказательство:
1) $\forall \lambda\in K:\ \ \ \lambda 0=0$.
2) $\forall \lambda\in K\ \ \ \forall a\in V:\ \ \ \lambda(-a)=-\lambda a$.
3) $\forall a\in V:\ \ \ 0 a= 0$.
4) $\forall a\in V:\ \ \ (-1)a=-a$.
