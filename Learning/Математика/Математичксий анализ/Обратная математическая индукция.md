**Утверждение.** Пусть $n$ - натуральное число и $P(m)$ - свойство натуральных чисел такое, что из верности $P(m\smash+\smash+)$ следует верность $P(m)$. Если $P(n)$ - истенно, тогда для всех $m\leqslant n$ свойство $P(m)$ тоже верно.
**Доказательство.** Докажем при помощи [[Доказательства по индукции|обычной индукции]] по $n$.
Пусть $n=1$, тогда по условию $P(1)$ должно быть верным, тогда для всех $m\leqslant n = 1$ (т.е. $m$ может быть равным только $1$) свойство $P(m)$ истенно. База индукции доказана.
Пусть теперь утверждение верно для $n$, т.е. если $P(n)$ - истенно, то $P(m)$ истенyо для всех $m\leqslant n$. Докажем это для $n\smash+\smash+$. Пусть $P(n\smash+\smash+)$ истенно, тогда по условию теоремы $P(n)$ тоже истенно, а значит, по предположению индукции $P(m)$ истенно для всех $m\leqslant n$. Но свойство $P(n\smash+\smash+)$ также истенно, а значит $P(m)$ верно для всех $m\leqslant n\smash+\smash+$, что завершает индукцию.
