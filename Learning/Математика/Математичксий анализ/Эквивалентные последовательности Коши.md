
Нам нужно определиться какие пределы [[Последовательности Коши рациональных чисел|последовательностей Коши]] считать равными без использования вещественных чисел (которые и являются их пределами). Для этого определим понятие эквивалентных последовательностей, чтобы в будущем говорить, что их пределы, т.е. действительные числа, равны, и на оборот.

**Определение.** Мы говорим, что две [[Последовательности и подпоследовательности точек множества|последовательности]] [[Построение рациональных чисел|рациональных чисел]] $\{a_i\}_{i=1}^\infty$ и $\{b_i\}_{i=1}^\infty$ являются $\varepsilon-$близкими, если $a_i$ и $b_i$ [[Эпсилон-близкие рациональные числа|эпсилон-блзики]] для всех $i\in\mathbb N$, т.е. $|a_i-b_i|<\varepsilon$ для всех $i$.

**Определение.** Пусть $\{a_i\}_{i=1}^\infty$ и $\{b_i\}_{i=1}^\infty$ две последовательности и $\varepsilon>0$, тогда последовательности называются конечно $\varepsilon$-близкими, если начианя с некоторого номера они $\varepsilon$-близки, т.е. существует $N\in\mathbb N$, что $|a_i-b_i|<\varepsilon$ для $i\geqslant N$.

**Определение.** Мы говорим, что две последовательности эквивалентны, если для всех $\varepsilon >0$ они конечно $\varepsilon$-близки. Другими словами:
$$\forall\varepsilon>0\ \ \ \exists N\in\mathbb N\ \ \ \forall n\geqslant N\ \ \ |a_n-b_n|<\varepsilon.$$

**Лемма 1.** Если две последовательности эквивалентны, то либо они обе являются [[Последовательности Коши рациональных чисел|последовательностями Коши]], либо они обе таковыми не являются.
**Доказательство.** Пусть $\{a_i\}_{i=1}^\infty$ - последовательность Коши и $\{b_i\}_{i=1}^\infty$ - эквивалентная ей, покажем, что $\{b_i\}_{i=1}^\infty$ тоже является последовательностью Коши. Возьмём некоторое $\varepsilon>0$, и для $\varepsilon/3$ найдём такое $N_1$, что $|a_j-a_k|<\varepsilon/3$ для всех $j,k\geqslant N_1$, и такое $N_2$, что $|a_i-b_i|<\varepsilon/3$ для всех $i\geqslant N_2$ (первое по определению последовательности Коши, второе по определению эквивалентной последовательности). Выберем наибольшее среди $N_1$ и $N_2$ и обозначим за $N$, тогда для всех $j,k\geqslant N$ будут выполнятся следующие неравенства:
$$|b_j-a_j|<\varepsilon/3,\ \ \ \ \ |a_j-a_k|<\varepsilon/3,\ \ \ \ \ |a_k-b_k|<\varepsilon/3.$$
Наконец оценим модуль разницы произвольных членов последовательности $\{b_i\}$ начиная с номер $N$, для любых $j,k\geqslant N$ будет верно неравенство:
$$|b_j-b_k|=|(b_j-a_j)+(a_j-a_k)+(a_k-b_k)|\leqslant|b_j-a_j|+|a_j-a_k|+|a_k-b_k|<\varepsilon,$$
из чего и следует, что эквивалентная последовательность тоже сходится.
Таким образом, две эквивалентные последовательности либо обе являются последовательностями Коши, либо обе не являются ими, поскольку из того, что одна из них является последовательностью Коши следует что и другая является ею.

**Лемма 2.** Пусть $\{a_i\}_{i=1}^\infty$ и $\{b_i\}_{i=1}^\infty$ две конечно $\varepsilon-$близкие последовательности и $\varepsilon>0$. Одна из них [[Последовательности Коши рациональных чисел|ограниченна]] тогда и только тогда, когда другая ограничена.
**Доказательство.** Пусть последовательность $\{a_i\}_{i=1}^\infty$ ограничена, тогда существует $M'$ такое, что $|a_i|\leqslant M'$ для всех возможных $i$. Найдём $N$, что $n\geqslant N$ будет $|a_n-b_n|<\varepsilon$, поскольку эти последовательность $\varepsilon$ конечно близки, но конечная последовательность $a_n-b_n$ (это тоже последовательность) для $n<N$, таким образом вся последовательность ограничена каким-то $M''$ (конечная её часть и бесконечная ограничены), таким образом:
$$|b_i|\leqslant|b_i-a_i|+|a_i|\leqslant M'+M''\ \ \ \ \ \forall i\in\mathbb N,$$
таким образом каждый элемент $b_i$ ограничен $M=M'+M''$, что и требовалось доказать.
