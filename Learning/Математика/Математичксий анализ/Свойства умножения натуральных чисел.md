**Утверждение 1 (Коммутативность).** Пусть $n,\ m$ - натуральные числа, тогда  $n\times m=m\times n$.
**Доказательство.** Проведём доказательство [[Доказательства по индукции|индукцией]] по $n$ при произвольном фиксированном $m$.
Пусть $n=1$, тогда $1\times m=m$ по [[Умножение натуральных чисел|определению умножения]]. Но $m\times 1=m$ тоже. Действительно, покажем при помощи индукции по $m$. При $m=1$ равенство очевидно верно по определению $1\times 1= 1$. Предположим равенство выполнено для $m$, покажем что оно врено и для $m\smash+\smash+$:
$$(m\smash+\smash+)\times 1=m\times 1 + 1=m+1=m\smash+\smash+$$
чем завершается индукция по $m$ и заканчивается дказательство "Шага индукции". Предположим теперь, что $n\times m=m\times n$ верно, покажем, что это верно и для $n\smash+\smash+$. Действительно:
$$(n\smash+\smash+)\times m=n\times m+m=m\times n + m=m\times(n\smash+\smash+)$$
Но последнее равенство пока что не чем не обасновано. Покажем, что мы применять формулу $n\times(m\smash+\smash+)=n\times m + n$. Покажем индукцией по $n$. Пусть $n=1$ (база индукции), тогда $1\times(m\smash+\smash+)=1\times m + 1$ очевидно из определений. Пусть теперь верно равенство $n\times(m\smash+\smash+)=n\times m + n$ для некоторого $n$, покажем что остаётся верным для $n\smash+\smash+$. Действительно:
$$(n\smash+\smash+)\times(m\smash+\smash+)=n\times(m\smash+\smash+)+(m\smash+\smash+)=n\times m +n+(m\smash+\smash+)=$$
$$=n\times m +m+(n\smash+\smash+)=(n\smash+\smash+)\times m+ (n\smash+\smash+)$$
что заканчивает индукцию $n$, а также и доказательство.

**Утверждение 2 (Дистрибутивность).** Для любых натуральных чисел $a,\ b,\ c$ выполняется формула $a(b+c)=ab+ac$.
**Доказательство.** Воспользуемся индукцией по $a$. Для $a=1$, формула очевидна:
$$1\times(b+c)=b+c=1\times b+1\times c$$
Пусть теперь формула верна для $a$, докажем её для $a\smash+\smash+$:
$$(a\smash+\smash+)(b+c)=a(b+c)+(b+c)=ab+ac+b+c=(a\smash+\smash+)b+(a\smash+\smash+)c$$
Утверждение доказано.

**Утверждение 3 (Ассоциативность).** Для любых $a,\ b,\ c$ - натуральных чисел, верна формула $(a\times b)\times c=a\times (b\times c)$.
**Доказательство.** Проведём доказательство по индукции. Пусть $a=1$, тогда
$$(1\times b)\times c=b\times c=1\times(b\times c)$$
Теперь, пусть верно утверждение для $a$, докажем его для $a\smash+\smash+$:
$$((a\smash+\smash+)\times b)\times c=(a\times b +b)\times c=(a\times b)\times c + b\times c=$$
$$=a\times(b\times c)+b\times c=(a\smash+\smash+)\times(b\times c)$$
Что и завершает доказательство.

**Утверждение 4 (Умножение в неравенстве).** Пусть $a,\ b,\ c$ натуральные числа. Тогда, если $a<b$, то $ac<bc$.
**Доказательство.** По [[Порядок натуральных чисел|определению порядка]], $b=d+a$, где $d$ - некоторое натуральное число. Тогда:
$$bc=(d+a)c=dc+ac$$
т.к. $dc$ - число натуральное, то $bc>ac$ по определению.

**Утверждение 5 (Закон сокращения умножения).** Пусть $a,\ b,\ c$ - натуральные числа. Тогда, если $ac=bc$, то $a=b$.
**Доказательство.** Из [[Порядок натуральных чисел|трихотомии натуральных чисел]] имеем либо $a>b$, либо $a=b$ либо $a<b$. Если $a>b$, то по предыдущему утверждению $ac>bc$, что противоречит условию. Аналогично, если $a>b$. Значит возможен только один вариант $a=b$.
