Здесь, рассмотрим ряд важных утверждений, касающихся [[Сложение натуральных чисел|сложения натуральных чисел]].

**Утверждение 1.** Если $n,\ m$ - натуральные числа, то $n+m$ тоже число натуральное.
**Доказательство.** Докажем [[Доказательства по индукции|индукцией]] по $n$, а $m$ - произвольное натуральное число:
1. **"База индукции".** Пусть $n = 1$, тогда $1+m=m\smash+\smash+$ - натуральное число по [[Построение натуральных чисел. Аксиомы Пеано|аксиоме 2]], т.к. $m$ - натуральное число по условию.
2. **"Шаг индукции".** Пусть для $n$ уже доказано, что $n+m$ - натуральное число, тогда для $n\smash+\smash+$ имеем, что $(n\smash+\smash+)+m=(n+m)\smash+\smash+$ - натуральное число (аналогично предыдущему пункту).
Тогда для любого натурального числа $n$, сложение является натуральным числом. Число $m$ изначально предпологалось произвольным натуральным числом.

**Лемма 1.** Если $n$ - произвольное натуральное число, то $n+1=n\smash+\smash+$.
**Доказательство.** Докажем [[Доказательства по индукции|индукцией]] по $n$:
1. **"База индукции".** Пусть $n = 1$, тогда $1+1=1\smash+\smash+$, что и требовалось показать.
2. **"Шаг индукции".** Пусть для $n$ утверждение верно (т.е. $n+1=n\smash+\smash+$), тогда для $n\smash+\smash+$ имеем:
$$(n\smash+\smash+)+1=(n+1)\smash+\smash+=(n\smash+\smash+)\smash+\smash+$$
Теперь, по [[Построение натуральных чисел. Аксиомы Пеано|аксиоме 5]], можем утверждать что равенство верно для всех натуральных чисел.

**Лемма 2.** Для всех натуральных чисел $n,\ m$ выполнено $n+(m\smash+\smash+)=(n+m)\smash+\smash+$.
**Доказательство.** Пусть $m$ - произвольное фиксированное натуральное число. Тогда при $n=1$ равенство верно, действительно:
$$1+(m\smash+\smash+)=(m\smash+\smash+)\smash+\smash+=(1+m)\smash+\smash+$$
Предположим, что верно $n+(m\smash+\smash+)=(n+m)\smash+\smash+$. Покажем, что при $n\smash+\smash+$ утверждение тоже выполняется:
$$(n\smash+\smash+)+(m\smash+\smash+) = (n+(m\smash+\smash+))\smash+\smash+=((n+m)\smash+\smash+)\smash+\smash+ = ((n\smash+\smash+)+m)\smash+\smash+$$
что завершает индукцию.

**Утверждение 2 (Коммутативный закон).** Если $n,\ m$ - натуральные числа, то $$n+m=m+n$$
**Доказательство.** Пусть $m$ - произовльное фиксированное натуральное число. Если $n=1$, то по лемме 1:
$$1+m=m\smash+\smash+= m + 1$$
Теперь, пусть $n+m=m+n$ для какого-то натурального $n$, покажем, что выполняется $(n\smash+\smash+) + m = m + (n\smash+\smash+)$, используя лемму 2:
$$(n\smash+\smash+)+m=(n+m)\smash+\smash+=(m+n)\smash+\smash+=m+(n\smash+\smash+)$$
что завершает индукцию.

**Утверждение 3 (Ассоциативный закон).** Для любях натуральных чисел $a,\ b,\ c$, выполняется равенство:
$$(a+b)+c=a+(b+c)$$
**Доказательство.** Пусть $a,\ b$ - произвольные фиксированные натуральные числа. Путь $c=1$, тогда:
$$(a+b)+1=(a+b)\smash+\smash+=a+(b\smash+\smash+)=a+(b+1)$$
Если для $c$ утверждение доказано, докажем его для $c\smash+\smash+$:
$$(a+b)+(c++)=((a+b)+c)\smash+\smash+=(a+(b+c))\smash+\smash+=$$$$=a+((b+c)\smash+\smash+)=a+(b+(c\smash+\smash+))$$
что завершает доказательство.

**Утверждение 4 (Закон сокращения).** Пусть $a,\ b,\ c$ - натуральные числа, такие, что $a+b=a+c$, то $b=c$.
**Доказательство.** Пусть $b,\ c$ - произвольные фиксированные натуральные числа. Путь $a=1$, тогда, если $1+b=1+c$, то $b\smash+\smash+=c\smash+\smash+$. В силу [[Построение натуральных чисел. Аксиомы Пеано|аксиомы 4]], получаем что $b=c$.
Если для $a$ утверждение доказано, докажем его для $a\smash+\smash+$. Пусть верно то, что $(a\smash+\smash+)+b=(a\smash+\smash+)+c$, из чего следует, что $(a+b)\smash+\smash+=(a+c)\smash+\smash+$. Снова по аксиоме 4, получаем, что обязательно должно выполнятся $a+b=a+c$. Теперь, по индукционному предположению, $b=c$.
