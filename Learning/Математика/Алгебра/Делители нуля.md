
**Определение.** В [[Кольца|кольце]] $K$ делителем нуля называется ненулевой элемент $a$ для которого есть ненулевой элемент $b$, что их произведение равно нулю, т.е. либо $ab=0$, либо $ba=0$ (в общем случае [[Кольца|кольцо не коммтативно по умножению]]). Если в кольце нет делителей нуля, то такое кольцо мы называем *"кольцом без делителей нуля"*.

Таким образом в кольце без делителей нуля выполняются следующие утверждения:
$$ab=0\Rightarrow a=0\ \ \ или\ \ \ b=0,$$
а также (что эквивалентно):
$$a\neq 0\ \ \ и\ \ \ b\neq 0\Rightarrow ab\neq 0.$$

В кольцах без делителей нуля возможно сокращение:
$$ac=bc\ \ \ и\ \ \ c\neq0\Rightarrow a=b.$$
Действительно, если $ac=bc$, то $ac-bc=0$ и $(a-b)c=0$. Так как $c\neq0$, то обязательно должно выполняться $a-b=0$, тогда $a=b$.
