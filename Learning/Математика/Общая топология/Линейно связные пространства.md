
**Определение.** Топологическое пространство называется *линейно связным*, если между каждыми двумя его точками можно провести [[Путь в топологическом пространстве|путь]].

**Примеры.**
* Линейно связным будет пространство $(I,\Omega_I)$, где $I=[0,1]\subseteq\mathbb R$. Действительно, для любых двух точек $a,b\in I$ путём соединяющих их будет функция $u(t)=a+t(b-a)$, поскольку она непрерывна (знание из матана), $u(0)=a$ и $u(1)=b$.
* В евклидовом пространстве любой размерности, аналогично для точек $\overline a,\overline b\in E^n$ будет функция $u(t)=\overline a+t(\overline b-\overline a)$.
