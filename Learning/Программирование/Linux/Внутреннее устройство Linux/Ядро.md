Ядро крутится покруг [[Что такое память?|памяти]]. Одной из его задач является разделение памяти на множество подразделови поддерждаение информации о них. Каждый [[Процессы|процесс]] получает себе отделный кусочек памяти, и ядро должно следить за тем, чтобы это всё поддерживалось.

Деятельность ядра можно разбить на 4 основные группы:
1. **[[Управление процессами|Процессы]]**. Поддержание процессов, передача доступа к ядру
2. **[[Управление памятью|Память]]**. Отслеживание всей памяти выданной для процессов.
3. **[[Драйвера устройств|Драйвера]]**. Поддержание связи между железом и процесами.
4. **[[Системные вызовы|Системные вызовы]].** Процессы часто используют *системные вызовы* для коммуникации с ядром.
