[[Ядро]] отвечает за смену между [[Процессы|процессами]] ***(context switch)*** во время их выполнения CPU. В общих чертах рассмотрим как это происходит:
1. Процессор останавливает выполнение процесса основываясь на своём личном таймере и предоставляет контроль ядру, переключаясь в режим для ядра.
2. Ядро записывает конечное состояние (последовательности 0 и 1) CPU в памяти, которые будут необходимы для возобновления данного процесса.
3. Ядро выполняет все задачи, которые пришли во время выполнения процесса (сбор информации из ввода и вывода и т.д.)
4. Ядро анализирует список процессов и выбирает из них один, готоый к запуску.
5. Ядро подготавливает память для нового процесса.
6. Ядро сообщает время выполнения очередного процесса CPU.
7. Ядро переключает режим CPU на пользовательский и передаёт управление процессору.

По итогу, само ядро работает между сменами процессов.
