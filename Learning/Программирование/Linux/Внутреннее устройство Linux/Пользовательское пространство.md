
Как говорилось [[Основные слои Linux|раньше]], [[ядро]] выделяет основную память для [[Процессы|процессов]] в пользовательском пространстве.
Большинство действий в Linux происходит в пользовательском пространстве. Хоть для ядра все процессы одинаковы, но сами по себе они выполняют разные задачи для пользователя. Есть процессы, с которыми мы напрямую работаем, естьбазовые процессы для работы системы, есть разнообразные утилиты и все они могут друг с другом коммуницировать.
