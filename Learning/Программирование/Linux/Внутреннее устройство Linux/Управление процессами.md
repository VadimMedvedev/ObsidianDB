Управление [[Процессы|процессами]] описывает начало, остановку, возобновление, расписание и прекращение процессов.

В современных операционных системах множество процессов происходят "одновременно".
Представим себе одноядерный процессор. Несколько процессов могут использовать CPU, но в каждый отдельный момент времени процессор обрабатывает только один процесс.
На практике, каждый процесс использует CPU на очень маленькое время ***(time slice)***, затем другой процесс также использует CPU на очень маленький период времени и т.д.
Каждый кусочек времени предоставляемый процессам достаточен для выполнения большого количества вычислений, но по итогу всё выполняется так быстро, что создаётся впечетление, что процессы выполняются одновременно.

В случае процессоров с несколькими ядрами (которых сейчас большинство) дела обстоят куда сложнее, в силу того что теперь процессор может выполнять несколько задач одновременно (теперь уже без ковычек).
