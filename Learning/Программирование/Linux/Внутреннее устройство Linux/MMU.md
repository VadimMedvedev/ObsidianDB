К счастью для [[Ядро|ядра]], в современных процеесорах добавляются MMU (memory managemant unit), которые позволяют использовать *виртуальную память*.
Во время использования виртуальной памяти, [[Процессы|процесс]] не знает конкретное место нахождения данных в [[Что такое память?|памяти]]. Заместо этого, процесс выполняется так, как будто запущен на отдельной машине для него.
Для этого, когда процесс берёт информацию с диска, этот сигнал перехвачивает MMU и преобразует его из сигнала с адрессом с точки зрения процесса, в настоящий адресс в компьютере. Для этого используются *карты адрессации* ***(memory address map)***.
Ядро же, должно создавать, постоянно поддерживать и изменять карты адрессации.
Например при [[Смена процессов|смене процесса]], ядро должно поменять карту адрессации одного процесса, на карту адрессации другого.
