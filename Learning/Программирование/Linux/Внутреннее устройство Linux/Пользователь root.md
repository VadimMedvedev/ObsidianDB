Но, помимо [[Пользователи|пользователей]] соответсвующих реальным людям, есть и другие. Их много, и одним из основных является пользователь root. Пользователь root может прекращать все процессы и изменять их, он может получить доступ к любому файлу в системе (что противоречит [[Пользователи|обычным правилам для пользователей]]). Таким образом, он известен как *суперпользователь*.
А пользователь, который может работать под root'ом, называется *администратором*.
