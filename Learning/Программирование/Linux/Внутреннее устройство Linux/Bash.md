Есть множество разннобразных [[shell]] программ, но все они происходят от Bourne Shell (разработанная Стивеном **Борном**).
Linux же использует улучшенную версию Bourne Shell названную *bash* (“Bourne-again” shell). Bash это основной shell для большенства дистрибутивов Linux, и в большенстве случаев он находится по ссылке /bin/sh.
