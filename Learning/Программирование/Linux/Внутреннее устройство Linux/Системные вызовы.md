[[Ядро]] предоставляет также несколько своих "услуг" для [[Процессы|пользовательских процессов]].
Например системные вызовы ***(system calls, syscalls)*** предназначены для выполнения задач, которые процеесы делают плохо или совсем не могут выполнить. Например открытие, чтение и запись в файл польностью включает в себя системные вызовы.
