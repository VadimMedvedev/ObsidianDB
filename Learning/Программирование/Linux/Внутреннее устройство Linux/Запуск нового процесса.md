Следующие два [[Системные вызовы|системных вызова]] важны для понимания как запускаются [[процессы]].

* fork() - когда процесс вызывает fork(), [[ядро]] создаёт копию данного процесса.
* exec() - Когда процесс вызывает exec(program), ядро загружает и запускает program, заменяя настоящий процесс.

Каждовый новый пользовательский процесс в Linux начинается с fork(), а затем открытия конкретного процесса при помощи exec(). 
Например при вводе в консоль команды ls, процесс shell, который запущен за терминальным окном, вызывает fork(), а затем его копия вызывает exec(ls).
