
[[Ядро]], при [[Управление процессами|смене процессов]], должно также работать с [[Что такое память?|памятью]]. При этом должно выполнятся следующие требования:
* У ядра должна быть собственная приватная память, доступ к которой, пользовательские процессы не смогут получить.
* Для каждого отдельного процесса должна быть выделенная область в памяти.
* Один процесс не может получить доступ к памяти другого процесса.
* Процессы могут делится памятью.
* Некоторая память в процессе может быть предназначена только для чтения (т.е. нужен запрет на запись).
* Система может использовать свободную память на диске помимо выделенной для неё.
