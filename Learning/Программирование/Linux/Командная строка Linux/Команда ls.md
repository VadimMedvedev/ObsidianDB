Команда ls выводит список содержимого директории. 
По умолчанию, это директория в которой находится пользователь, но можно указать и другую директории или даже файл (выведится информация об этом файле), написав её в аргументе.
Также у команды ls есть много полезных ключей. Например:
* -l - для дитального вывода файлов в директории
* -F - для вывода типов файлов
