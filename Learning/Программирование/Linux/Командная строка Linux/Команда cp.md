В самой простой форме, cp копирует содержимое file1 в file2, для этого надо ввести:
``` Bash
cp file1 file2
```

Можно также скопировать файл в другую директорию, оставляя прежнее название:
``` Bash
cp file dir
```

Для того, чтобы скопировать сразу несколько файлов, можно написать:
``` Bash
cp file1 file2 file3 dir
```
