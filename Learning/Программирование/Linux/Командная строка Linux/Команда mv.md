Команда mv (move) перемещает файл (т.е. посути [[Команда cp|копирует]], и удаляет старый файл). В самой простой форме, она переименовывает файл (т.е. пермещает в ту же папку, но с другим именем):
``` Bash
mv file1 file2
```

Можно перемещать файл в разные директории как и в случае [[Команда cp|команды cp]].
