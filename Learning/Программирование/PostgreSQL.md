
### Установка

Устанавливаем пакет postgresql (можно также какой-то определённой версии, иногда общая версия не хочет работать).
Включаем процесс postresql.service через systemctl, но часто для этого сначала нужно инициализовать базу данных. Это можно сделать командой postgresql-setup initdb.
Теперь настроим доступ к нашей базе данных, для этого найдём файл pg_hba.conf:
``` Bash
sudo find / -name "pg_hba.conf"
```
Добавляем запись с какого адраса доступно подключение, к каким пользователям и базам данных (по умолчанию можно оставить all).

В той же директории в файле postgresql.conf изменяем тип аутентификации на md5 и указываем какие адреса мы будем слушать. При желании можем изменить порт.
```
listen_addresses = '*'                  # все интерфейсы

password_encryption = md5

port = 5432
```

Теперь заходим в psql (инструмент для работы с базами данных через командную строчку) под пользователем postgres
``` bash
sudo -iu postgres psql
```
Там мы создаём тестовую базу данных и пользователя, через которого мы будем входить в систему
```
CREATE DATABASE testdb;

CREATE USER pguser WITH PASSWORD 'password123';

GRANT ALL PRIVILEGES ON DATABASE testdb TO pguser;
```
(Выход из psql - \q).

В pgadmin добавляем сервер указывая имя (произвольное), а также ip адрес, имя пользователя (которого мы только что создавали) и его пароль.
